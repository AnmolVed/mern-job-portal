# MERN Job Portal

A full-stack job portal built using MERN stack.

## Features
- User Authentication (JWT)
- Role-based access (Employer / Job Seeker)
- Job Posting
- Apply to Jobs
- My Applications
- Real-time UI updates

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Setup

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev
